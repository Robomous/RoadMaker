# Archive — pre-reset roadmap (superseded 2026-07)

Everything in this folder is a **historical record**, superseded by the
[Road to Parity roadmap](../../README.md) in the 2026-07 roadmap reset
([ADR-0007](../../../decisions/0007-roadmap-reset-road-to-parity.md)).

- `roadmap.md`, `gap_analysis.md`, `asset_candidates.md` — the retired
  milestone/version roadmap (M1 … M5, v0.2.0 … v0.10.0 checkpoints) and its
  supporting analyses.
- `seeds/` — scope sketches for the retired milestones.
- `golden_scenes/` — the GS-1 … GS-4 golden-scene specs and baseline
  render. Retired; if visual scene benchmarks are wanted again, they will
  be re-derived from the new golden workflows.
- `golden_workflows/` — the original GW-1/GW-2 workflows and the v0.4.0
  gate record. Replaced by the new
  [golden-workflow set](../../golden_workflows/README.md).
- `releases/` — notes for the retired checkpoint releases, whose tags and
  GitHub releases were deleted in the reset.

Nothing here is normative. Links within this folder may point at other
archived pages; links out of it were left valid at archive time but are not
maintained.
